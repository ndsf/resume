\heading{Research Experience}
\entry{0em}{Xr}{{\bfseries Shanghai Jiao Tong University} & Sep 2021 - Present}
\entry{2em}{X}{Data-Driven Software Technology Lab \quad Ph.D. Student \quad Memory Storage Systems, Distributed Systems \quad Advisor: Prof. Linpeng Huang}
\entry{2em}{X}{%
    Participated in the project "\textbf{Disaggregated Memory Computing Framework Research}". Designed high-concurrency hash indexes for RDMA and CXL high-speed networks. Existing indexes are limited by the constrained functionality of network primitives, suffering from write position lookup, concurrency conflicts, resize waiting, and cache coherence overheads during writes, which degrade performance in high-concurrency and skewed scenarios. This project achieves high-performance hash indexing through redesigning index structures, leveraging RDMA features such as shared receive queues, counter-triggered compactions, and optimistic resize detection, implemented with RDMA and C++. Currently submitting first-author paper "Redesigning Hash Indexes on Disaggregated Memory for Minimal Network Overhead" \cite{chen2026rdma} to ICDE 2026 (CCF-A conference). \\
}
\entry{2em}{X}{%
    Participated in Huawei project "\textbf{High-Performance and High-Reliability File System for New Media}". Research on Linux kernel file systems based on openEuler's EulerFS for persistent CPU cache environments. Published first-author paper "Redesigning Data and Metadata Updates in PM File Systems with Persistent CPU Caches" \cite{chen2024redesigning} at DASFAA 2024 (CCF-B conference). This paper proposes that completely eliminating cache line flushes can cause write amplification due to random cache line evictions, and selects the most effective update mechanism by monitoring system calls and memory-mapped file access patterns. Published first-author paper "FusionFS: A Contention-Resilient File System for Persistent CPU Caches" \cite{chen2025fusionfs} in ACM TACO 2025 (CCF-A journal). This paper further categorizes and uses techniques such as cache allocation to mitigate various contentions faced by persistent CPU caches, implementing the Linux kernel file system FusionFS in C \url{https://github.com/SJTU-DDST/FusionFS}. \\
}
